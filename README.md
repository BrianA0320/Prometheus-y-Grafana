# Práctica de Monitoreo: Prometheus + Grafana con Docker

Este proyecto despliega un stack de monitoreo básico utilizando contenedores de Docker.

## Servicios incluidos:
- **Prometheus**: Base de datos de métricas.
- **Grafana**: Visualización de datos.
- **Node Exporter**: Extractor de métricas del sistema.

## Cómo ejecutarlo:
1. Clonar el repositorio.
2. Ejecutar `docker-compose up -d`.
3. Acceder a Grafana en `http://localhost:3000` (User/Pass: admin/admin).
